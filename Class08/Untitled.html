Q8 - Compare plots of PC1 vs PC2 to plot of PC1 vs PC3

The PC1 vs PC2 plot has a clearer separation between M and B samples than the PC1 vs PC3 plot. This shows that PC2 captures more class-separating structure than PC3. PC1 seems to drive most of the separation overall, while later components add less discriminatory power.

Variance per Principal Component

We want to compute the variance explained by each principal component from the PCA model.

pr.var <- wisc.pr$sdev^2
pve <- pr.var / sum(pr.var)
plot(pve, xlab="Principal Component",
     ylab="Proportion of Variance Explained",
     ylim=c(0,1), type="o")

Q9 — Plot Interpretation

The plot shows a steep drop in variance followed by a long tail. This shows that most of the meaningful structure in the data is captured by the first few PCs. Other later components contribute relatively less variance.

Hierarchical Clustering

We need to scale the data and compute a distance matrix for hierarchical clustering.

data.scaled <- scale(wisc.data)
data.dist <- dist(data.scaled)

After this we perform hierarchical clustering using complete linkage and plot the dendrogram.

wisc.hclust <- hclust(data.dist, method = "complete")
plot(wisc.hclust)
plot(wisc.hclust)
abline(h = 19, col = "red")

Split the dendrogram into two clusters using cutree().

wisc.clusters <- cutree(wisc.hclust, k = 2)
table(wisc.clusters, diagnosis)
             diagnosis
wisc.clusters   B   M
            1 357 210
            2   0   2

This table compares cluster assignments to the diagnosis labels. The match is imperfect because hierarchical clustering is unsupervised and does not use class labels.

We will do the hierarchical clustering again but using average linkage to compare results.

wisc.hclust.avg <- hclust(data.dist, method = "average")
plot(wisc.hclust.avg)
wisc.clusters.avg <- cutree(wisc.hclust.avg, k = 2)

Average Linkage Cluster Comparison

Using average linkage makes a similar result to complete linkage. The clusters still don’t align perfectly with diagnosis labels. This shows that hierarchical clustering w/o labels doesn’t perfectly separate B and M samples.
